# Verse 14: Three Dimensions of Underdetermination

## Level 1: Splitting (where are the word boundaries?)
- 55 split points → 54.5 billion decompositions
- Range: 2 to 26 names
- Determines: HOW MANY attributes the divine has

## Level 2: Etymology (what root does each word come from?)
- For the 9 traditional names: 3×4×4×3×3×3×4×4×4 = 82,944 combinations
- Each combination = a different theology from the SAME 9 words
- Determines: WHAT each attribute means

## Level 3: Combined
- 54.5 billion × 82,944 (minimum) = beyond computation
- Each path = a complete theological system
- The tradition picks ONE path. The text contains ALL.

## The bhāva example
Same three letters (bh-ā-va). Four theologies:
1. √bhū (to be) → Being itself → ONTOLOGICAL
2. √bhū (to become) → Becoming → PROCESS  
3. √bhā (to shine) → Radiance → LUMINOSITY
4. bhāva (mood) → Disposition → AESTHETIC

The commentator picks one. The pick IS the school.
